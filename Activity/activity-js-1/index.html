Activity

Your task is to create a simple calculator that can add, subtract, multiply, and divide two numbers. Here's what the calculator should do:

  1. User clicks on a button to start the calculator.
  2. Prompt the user to enter the first number.
  3. Prompt the user to enter the second number.
  4. Alert the user with the result of adding the two numbers.
  5. Alert the user with the result of subtracting the two numbers.
  6. Alert the user with the result of multiplying the two numbers.
  7. Alert the user with the result of dividing the two numbers.

Then, once you have the basic calculator working, add error checking to make sure the user enters valid numbers. If the user enters something that is not a number, alert the user with an error message and do not perform the calculation.
